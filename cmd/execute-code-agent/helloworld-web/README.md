# 炫酷Helloworld欢迎页面

这是一个为万秀彬同志创建的炫酷欢迎页面，包含现代前端技术和视觉效果。

## 功能特性

- 🎨 **炫酷视觉效果**：使用CSS3动画、渐变和阴影效果
- ✨ **粒子背景**：动态粒子效果，支持鼠标交互
- 📱 **响应式设计**：适配各种屏幕尺寸
- ⏱️ **2026倒计时**：实时显示距离2026年的倒计时
- 🎯 **交互效果**：鼠标悬停、点击特效
- 🌈 **多彩主题**：使用渐变色彩和动画效果

## 页面内容

1. **欢迎标题**：欢迎万秀彬同志的醒目标题
2. **祝福语**："祝福2026健康强壮，力能鼎牛"的炫酷展示
3. **特性展示**：健康、强壮、鼎牛三个特性的视觉呈现
4. **倒计时**：距离2026年的实时倒计时
5. **粒子背景**：动态交互式粒子效果

## 技术栈

- HTML5
- CSS3 (动画、渐变、Flexbox、Grid)
- JavaScript (粒子效果、倒计时、交互)
- Particles.js 粒子库
- Font Awesome 图标
- Google Fonts 字体

## 使用方法

1. 直接打开 `index.html` 文件在浏览器中查看
2. 或部署到Web服务器

## 交互功能

- 鼠标移动：卡片3D倾斜效果
- 点击祝福语：粒子爆炸特效
- 鼠标悬停粒子：粒子排斥效果
- 点击粒子：添加新粒子

## 浏览器兼容性

支持所有现代浏览器（Chrome, Firefox, Safari, Edge）
